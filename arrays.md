# Массивы в JavaScript: Полное руководство

Массив — это упорядоченная коллекция данных. В отличие от объектов, где данные хранятся в виде пар "ключ-значение", в массивах элементы располагаются по порядку и имеют числовые индексы, начиная с нуля.

---

## 1. Создание массива

Существует два основных способа создания массива:

```javascript
// 1. Литерал массива (самый частый способ)
const fruits = ["Яблоко", "Апельсин", "Слива"];

// 2. Конструктор Array
const numbers = new Array(1, 2, 3);
```

В массиве могут храниться элементы любых типов: числа, строки, объекты и даже другие массивы.

---

## 2. Доступ к элементам и свойство `length`

Элементы массива нумеруются с **0**.

```javascript
const colors = ["Red", "Green", "Blue"];

console.log(colors[0]); // "Red"
console.log(colors.length); // 3 (количество элементов)

// Изменение элемента
colors[1] = "Yellow";
```

---

## 3. Основные методы (Очередь и Стек)

JavaScript позволяет использовать массив как стек или очередь.

-   **push(...items)** — добавляет элементы в **конец**.
-   **pop()** — удаляет последний элемент и возвращает его.
-   **shift()** — удаляет первый элемент и возвращает его.
-   **unshift(...items)** — добавляет элементы в **начало**.

```javascript
let stack = [];
stack.push(1); // [1]
stack.push(2); // [1, 2]
stack.pop();    // возвращает 2, массив теперь [1]
```

---

## 4. Перебор массива

### Цикл `for` (классический)
```javascript
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
```

### Цикл `for...of` (современный)
```javascript
for (let fruit of fruits) {
  console.log(fruit);
}
```

### Метод `forEach`
Выполняет функцию для каждого элемента.
```javascript
fruits.forEach((item, index, array) => {
  console.log(`${item} имеет индекс ${index}`);
});
```

---

## 5. Поиск в массиве

-   **indexOf(item)** — ищет элемент и возвращает его индекс или -1.
-   **includes(item)** — возвращает `true`, если элемент есть в массиве.
-   **find(callback)** — возвращает **первый** элемент, удовлетворяющий условию.
-   **filter(callback)** — возвращает **новый массив** из всех элементов, прошедших проверку.

```javascript
const users = [
  { id: 1, name: "Ivan" },
  { id: 2, name: "Anna" }
];
const user = users.find(u => u.id === 2); // { id: 2, name: "Anna" }
```

---

## 6. Трансформация массива

Это самые мощные методы для работы с данными.

### map(callback)
Создает новый массив, применяя функцию к каждому элементу.
```javascript
const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2); // [2, 4, 6]
```

### reduce(callback, initialValue)
Вычисляет одно значение на основе всего массива (например, сумму).
```javascript
const sum = nums.reduce((acc, curr) => acc + curr, 0); // 6
```

---

## 7. Многомерные массивы

Массивы могут содержать другие массивы, образуя матрицы или таблицы.

```javascript
const matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];
console.log(matrix[1][1]); // 5 (центральный элемент)
```

---

## Лучшие практики

1.  **Используйте `const`** для объявления массивов (вы всё еще сможете изменять их содержимое).
2.  **Предпочитайте методы `map/filter/reduce`** вместо циклов `for`, так как они делают код более декларативным и чистым.
3.  **Не используйте `delete`** для удаления элементов (остается "дырка" `undefined`). Используйте `splice()`, `pop()` или `shift()`.
4.  **Будьте осторожны с `length = 0`** — это самый быстрый способ очистить массив, но он уничтожает данные безвозвратно.
