# Регулярные выражения в JavaScript: Полное руководство

**Регулярные выражения (Regex)** — это мощный инструмент для поиска, сопоставления и манипуляции текстом по определенным шаблонам. Они незаменимы при проверке email, поиске слов в тексте или замене определенных фрагментов данных.

---

## 1. Создание регулярного выражения

В JavaScript есть два способа создать объект регулярного выражения:

```javascript
// 1. Литерал (рекомендуется, если шаблон не меняется)
const regex1 = /шаблон/flags;

// 2. Конструктор (используется, если шаблон создается динамически)
const regex2 = new RegExp("шаблон", "flags");
```

---

## 2. Основной синтаксис

- **Символьные классы**:
  - `\d` — любая цифра.
  - `\w` — любая буква, цифра или подчеркивание.
  - `\s` — любой пробельный символ (пробел, табуляция).
  - `.` — любой символ (кроме переноса строки).
- **Якоря**:
  - `^` — начало строки.
  - `$` — конец строки.

---

## 3. Квантификаторы (Количество)

- `+` — один или более.
- `*` — ноль или более.
- `?` — ноль или один.
- `{n}` — ровно `n` раз.
- `{n,m}` — от `n` до `m` раз.

```javascript
const regex = /\d{3}-\d{2}-\d{2}/; // Шаблон для телефона типа 123-45-67
```

---

## 4. Флаги

Флаги меняют способ поиска:
- `i` — поиск без учета регистра (A и a — одно и то же).
- `g` — глобальный поиск (искать все совпадения, а не только первое).
- `m` — многострочный режим.

---

## 5. Основные методы

### regex.test(str)
Проверяет, есть ли совпадение в строке. Возвращает `true` или `false`.
```javascript
const isEmail = /@/.test("test@mail.com"); // true
```

### str.match(regex)
Возвращает массив совпадений.
```javascript
const text = "Цены: 100, 200, 300";
const prices = text.match(/\d+/g); // ["100", "200", "300"]
```

### str.replace(regex, replacement)
Заменяет найденные части на другие.
```javascript
const secretText = "Мой номер 123-456";
const masked = secretText.replace(/\d/g, "*"); // "Мой номер ***-***"
```

---

## 6. Группы и скобки

Круглые скобки `()` позволяют группировать части шаблона и извлекать их отдельно.

```javascript
const regex = /(\d+) (кг|гр)/;
const result = "Вес: 500 гр".match(regex);
console.log(result[1]); // "500"
console.log(result[2]); // "гр"
```

---

## Лучшие практики

1. **Не делайте регулярные выражения слишком сложными**. Если Regex занимает 3 строки, его будет почти невозможно отладить. Иногда проще написать несколько простых условий в коде.
2. **Используйте онлайн-тестеры** (например, regex101.com) для проверки своих шаблонов.
3. **Помните о производительности**. Плохо написанное регулярное выражение может сильно замедлить работу программы на больших текстах.
4. **Используйте именованные группы** в современном JS для лучшей читаемости: `(?<name>pattern)`.
