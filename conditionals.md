# Модуль 4: Условные операторы и выражения

В этом модуле мы научимся управлять логикой программы: заставлять код выполнять разные действия в зависимости от условий.

---

## 1. Конструкция `if`, `else if` и `else`

Это основной способ ветвления кода.

```javascript
let age = 18;

if (age >= 18) {
  console.log("Доступ разрешен");
} else if (age > 12) {
  console.log("Доступ ограничен");
} else {
  console.log("Доступ запрещен");
}
```

-   **Важно**: В скобках `if (...)` значение всегда приводится к логическому типу (Boolean). Как мы помним из Модуля 2, значения вроде `0`, `""`, `null`, `undefined` и `NaN` превратятся в `false`.

---

## 2. Логические операторы

Используются для создания сложных условий.

1.  **`&&` (И)**: Возвращает `true`, если оба операнда истинны.
2.  **`||` (ИЛИ)**: Возвращает `true`, если хотя бы один операнд истинный.
3.  **`!` (НЕ)**: Инвертирует значение.

### Короткое замыкание (Short-circuit evaluation)
JavaScript вычисляет логические выражения слева направо и останавливается, как только результат становится ясен:
- `false && ...` — правая часть не вычислится.
- `true || ...` — правая часть не вычислится.

---

## 3. Тернарный оператор `? :`

Это единственный оператор в JS, принимающий три операнда. Краткая форма `if...else`.

**Синтаксис**: `условие ? значение_если_true : значение_если_false`

```javascript
let message = (age >= 18) ? "Привет, взрослый!" : "Привет, малыш!";
```

---

## 4. Оператор `switch`

Используется, когда нужно сравнить одну переменную с множеством конкретных значений.

```javascript
let fruit = "Apple";

switch (fruit) {
  case "Apple":
    console.log("Это яблоко");
    break; // Останавливает выполнение, иначе код пойдет в следующий case
  case "Banana":
    console.log("Это банан");
    break;
  default:
    console.log("Неизвестный фрукт");
}
```

-   **Важно**: `switch` использует **строгое сравнение (`===`)**.
-   **Break**: Если забыть `break`, выполнение продолжится в следующем `case`, даже если условие не совпадает (это называется "fall-through").

---

## 5. Оператор нулевого слияния `??`

Современный оператор, который возвращает правый операнд, только если левый равен `null` или `undefined`.

```javascript
let name = null;
let displayName = name ?? "Аноним"; // "Аноним"

let count = 0;
let result = count ?? 10; // 0 (так как 0 — не null и не undefined)
// Сравните с ||
let badResult = count || 10; // 10 (так как 0 — это falsy)
```

---

## Резюме
- Используйте `if` для диапазонов и сложных логических условий.
- Используйте `switch` для выбора из набора конкретных значений.
- Тернарный оператор идеален для простых присваиваний по условию.
- `??` безопаснее, чем `||`, если вам нужно учитывать `0` или пустые строки как валидные значения.
