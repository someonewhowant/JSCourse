# Canvas API в JavaScript: Полное руководство

**Canvas API** предоставляет способ рисования 2D-графики с помощью JavaScript. Его можно использовать для создания графиков, анимаций, игр и обработки изображений на лету.

---

## 1. Элемент `<canvas>`

Сначала нужно добавить элемент в HTML. У него обязательно должны быть атрибуты `width` и `height` (через CSS задавать размер не рекомендуется, так как это просто растянет картинку).

```html
<canvas id="myCanvas" width="400" height="200"></canvas>
```

---

## 2. Контекст рисования

Чтобы начать рисовать, нужно получить "контекст" в JavaScript.

```javascript
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d'); // Получаем 2D-контекст
```

---

## 3. Рисование прямоугольников

Это единственная фигура, которая рисуется сразу одним методом.

- `fillRect(x, y, w, h)` — закрашенный прямоугольник.
- `strokeRect(x, y, w, h)` — только контур.
- `clearRect(x, y, w, h)` — стирает область (делает прозрачной).

```javascript
ctx.fillStyle = 'green'; // Цвет заливки
ctx.fillRect(10, 10, 100, 50);
```

---

## 4. Пути (Линии и сложные фигуры)

Для рисования других фигур нужно использовать "пути".

```javascript
ctx.beginPath();       // Начало пути
ctx.moveTo(50, 50);    // Передвинуть "перо" в точку
ctx.lineTo(150, 50);   // Линия до точки
ctx.lineTo(100, 150);  // Линия дальше
ctx.closePath();       // Замкнуть фигуру (назад к началу)

ctx.strokeStyle = 'red';
ctx.stroke();          // Обводка
// ctx.fill();         // Или заливка
```

---

## 5. Рисование текста

```javascript
ctx.font = '30px Arial';
ctx.fillText('Привет, Canvas!', 10, 50);
ctx.strokeText('Только контур', 10, 100);
```

---

## 6. Работа с изображениями

Вы можете рисовать на холсте картинки, загруженные извне.

```javascript
const img = new Image();
img.src = 'picture.jpg';
img.onload = () => {
  ctx.drawImage(img, 0, 0, 100, 100);
};
```

---

## 7. Анимация: `requestAnimationFrame`

Для создания плавной анимации используется метод `requestAnimationFrame`, который просит браузер вызвать функцию перед следующей перерисовкой экрана (обычно 60 раз в секунду).

```javascript
let x = 0;
function draw() {
  ctx.clearRect(0, 0, canvas.width, canvas.height); // Чистим холст
  ctx.fillRect(x, 50, 50, 50); // Рисуем в новой позиции
  x += 2;
  requestAnimationFrame(draw); // Зацикливаем
}
draw();
```

---

## Лучшие практики

1. **Всегда чистите холст** (`clearRect`) перед каждым кадром анимации.
2. **Используйте `beginPath()`** перед рисованием каждой новой независимой фигуры, иначе браузер будет "тянуть" старые линии за новыми.
3. **Оптимизируйте отрисовку**. Не делайте тяжелых вычислений внутри цикла анимации.
4. **Помните о разрешении**. На экранах Retina (`devicePixelRatio > 1`) обычный холст может выглядеть мутным. Его нужно масштабировать программно.
