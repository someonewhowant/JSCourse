# Функции в JavaScript: Полное руководство

Функции — это "кирпичики" программы. Они позволяют объединять повторяющиеся действия в один блок кода, которому можно дать имя и вызывать его по мере необходимости.

---

## 1. Объявление функции (Function Declaration)

Самый классический способ создания функции. Такие функции "всплывают" (hoisting), их можно вызывать до места их фактического объявления в коде.

```javascript
function sayHello(name) {
  return `Привет, ${name}!`;
}

console.log(sayHello("Иван")); // "Привет, Иван!"
```

---

## 2. Функциональное выражение (Function Expression)

Функция создается как значение и присваивается переменной. Они не всплывают.

```javascript
const sayGoodbye = function(name) {
  return `Пока, ${name}!`;
};
```

---

## 3. Стрелочные функции (Arrow Functions)

Появились в ES6. Они имеют более краткий синтаксис и не имеют своего контекста `this`.

```javascript
const add = (a, b) => a + b;

// Если тело функции сложное, нужны фигурные скобки и return
const multiply = (a, b) => {
  const result = a * b;
  return result;
};
```

---

## 4. Параметры и аргументы

### Параметры по умолчанию
Если аргумент не передан, используется значение по умолчанию.
```javascript
function greet(name = "Гость") {
  console.log(`Привет, ${name}`);
}
greet(); // "Привет, Гость"
```

### Остаточные параметры (Rest parameters)
Позволяют собирать неопределенное количество аргументов в массив.
```javascript
function sumAll(...args) {
  return args.reduce((acc, n) => acc + n, 0);
}
console.log(sumAll(1, 2, 3, 4)); // 10
```

---

## 5. Возврат значения (return)

Директива `return` прекращает выполнение функции и возвращает значение. Если `return` отсутствует, функция возвращает `undefined`.

```javascript
function checkAge(age) {
  if (age >= 18) {
    return true;
  }
  return false;
}
```

---

## 6. Область видимости и замыкания

Функция имеет доступ к переменным, объявленным снаружи неё.

- **Локальные переменные**: объявлены внутри функции, недоступны снаружи.
- **Внешние переменные**: объявлены вне функции, доступны внутри.
- **Замыкание**: это способность функции запоминать свою внешнюю область видимости даже после того, как внешняя функция завершила выполнение.

---

## 7. Функции обратного вызова (Callbacks)

Функция, которая передается в другую функцию как аргумент для последующего вызова.

```javascript
function processInput(callback) {
  const name = "Алексей";
  callback(name);
}

processInput((n) => console.log(`Данные получены для: ${n}`));
```

---

## Лучшие практики

1.  **Имя функции должно быть глаголом**. Оно должно описывать действие: `showElement`, `calculateTotal`, `fetchData`.
2.  **Одна функция — одна задача**. Если функция делает слишком много, разбейте её на несколько маленьких.
3.  **Избегайте побочных эффектов**. В идеале функция должна зависеть только от своих аргументов и возвращать предсказуемый результат.
4.  **Используйте стрелочные функции для коротких колбэков**, это делает код чище.
