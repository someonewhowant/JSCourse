# Объекты в JavaScript: Полное руководство

Объекты используются для хранения коллекций различных данных и более сложных сущностей. В то время как массивы упорядочены по индексам, объекты хранят данные в виде пар "ключ: значение".

---

## 1. Литерал объекта

Это самый простой и популярный способ создания объекта.

```javascript
const user = {
  name: "Ivan",
  age: 30,
  "likes javascript": true // Ключ из нескольких слов должен быть в кавычках
};
```

### Доступ к свойствам
```javascript
console.log(user.name); // "Ivan" (через точку)
console.log(user["likes javascript"]); // true (через квадратные скобки)
```

---

## 2. Добавление и удаление свойств

Объекты в JavaScript динамичны.

```javascript
user.isAdmin = true; // Добавление
delete user.age;    // Удаление
```

---

## 3. Методы объектов и `this`

Метод — это функция, которая является свойством объекта.

```javascript
const calculator = {
  brand: "Casio",
  add(a, b) {
    return a + b;
  },
  showBrand() {
    // 'this' ссылается на текущий объект
    console.log(this.brand);
  }
};

calculator.showBrand(); // "Casio"
```

**Важно:** Стрелочные функции не имеют своего `this`. Если использовать стрелочную функцию как метод, `this` будет взят из внешней области видимости.

---

## 4. Перебор свойств объекта

Для перебора всех ключей объекта используется цикл `for...in`.

```javascript
for (let key in user) {
  console.log(`Ключ: ${key}, Значение: ${user[key]}`);
}
```

---

## 5. Полезные методы встроенного объекта `Object`

JavaScript предоставляет статические методы для работы с объектами:

- **Object.keys(obj)** — возвращает массив ключей.
- **Object.values(obj)** — возвращает массив значений.
- **Object.entries(obj)** — возвращает массив пар [ключ, значение].
- **Object.assign(dest, ...src)** — копирует свойства из одного или нескольких объектов в целевой объект.

```javascript
const keys = Object.keys(user); // ["name", "likes javascript", "isAdmin"]
```

---

## 6. Клонирование и копирование

Присвоение объекта переменной копирует не сам объект, а **ссылку** на него.

```javascript
const admin = user; // Оба ссылаются на один и тот же объект в памяти
```

Чтобы создать настоящий клон (поверхностный), можно использовать:
1. `Object.assign({}, user)`
2. Спред-оператор: `const clone = { ...user };`

---

## 7. Опциональная цепочка `?.`

Позволяет безопасно обращаться к глубоко вложенным свойствам, даже если промежуточные объекты не существуют.

```javascript
const val = user.address?.street; // вернет undefined, а не ошибку, если address нет
```

---

## Лучшие практики

1. **Используйте короткое свойство**. Если имя ключа совпадает с именем переменной: `{ name: name }` -> `{ name }`.
2. **Проверяйте наличие свойства** через оператор `in` или метод `hasOwnProperty()`.
3. **Используйте `const`** для объявления объектов. Вы сможете менять их свойства, но не сможете переназначить саму переменную.
4. **Для глубокого клонирования** используйте `structuredClone(obj)` (современный стандарт).
