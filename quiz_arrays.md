# Квиз: Массивы в JavaScript

Проверьте свои знания о массивах. Ответы в конце.

---

### Вопрос 1
Какой индекс имеет первый элемент массива?

A) `1`  
B) `-1`  
C) `0`  
D) Зависит от длины массива  

---

### Вопрос 2
Какой метод добавляет один или несколько элементов в **начало** массива?

A) `push()`  
B) `unshift()`  
C) `shift()`  
D) `pop()`  

---

### Вопрос 3
Что выведет этот код?

```javascript
const arr = [1, 2, 3, 4, 5];
const filtered = arr.filter(n => n > 3);
console.log(filtered);
```

A) `[4, 5]`  
B) `[1, 2, 3]`  
C) `4`  
D) `true`  

---

### Вопрос 4
Какой метод массива создает новый массив, преобразуя каждый элемент исходного массива с помощью функции?

A) `forEach()`  
B) `find()`  
C) `reduce()`  
D) `map()`  

---

### Вопрос 5
Как правильно узнать количество элементов в массиве `myArr`?

A) `myArr.size()`  
B) `myArr.length`  
C) `count(myArr)`  
D) `myArr.elements`  

---

## Ответы

<details>
<summary>Нажмите, чтобы увидеть ответы</summary>

1. **C) 0** (Массивы в JS имеют нулевой индекс).
2. **B) unshift()** (`push` добавляет в конец, `shift` удаляет из начала).
3. **A) [4, 5]** (Метод `filter` возвращает новый массив со всеми элементами, прошедшими проверку).
4. **D) map()** (Это основной метод для трансформации данных в массиве).
5. **B) myArr.length** (Это свойство, а не метод, поэтому скобки не нужны).

</details>
