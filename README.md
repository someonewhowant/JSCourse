# Полный курс по JavaScript (Core + Browser + Architecture)

Добро пожаловать в структурированный репозиторий для изучения современного JavaScript. Курс охватывает всё: от самых основ до продвинутых тем архитектуры и работы с браузерным окружением.

---

## 🚀 Как пользоваться курсом

1.  **Теория**: Начинайте с `introduction.md` и двигайтесь по списку модулей.
2.  **Практика**: После каждого теоретического блока проходите квиз в формате Markdown.
3.  **Для преподавателей**: В репозитории есть файлы `.gift` для импорта тестов в LMS (Moodle, Canvas и др.).
4.  **Финальный проект**: В конце вас ждет полноценное ТЗ на разработку приложения.
