# Модуль 0: Введение в JavaScript

Добро пожаловать в курс по JavaScript! В этом модуле мы разберем, что это за язык, почему он так популярен и как будет устроено наше обучение.

---

## 1. Что такое JavaScript?

**JavaScript (JS)** — это высокоуровневый, мультипарадигменный язык программирования. Первоначально созданный для того, чтобы сделать веб-страницы "живыми", сегодня он является одним из самых мощных и универсальных инструментов в мире IT.

### Где используется JS сегодня?
1.  **Front-end**: Создание интерфейсов сайтов (React, Angular, Vue).
2.  **Back-end**: Написание серверной части приложений (Node.js).
3.  **Mobile**: Разработка мобильных приложений (React Native, Ionic).
4.  **Desktop**: Программы для ПК (Electron — так сделаны VS Code, Discord, Slack).

---

## 2. Краткая история

-   **1995 год**: Брендан Эйх создает первую версию языка всего за 10 дней для браузера Netscape Navigator. Изначально язык назывался *Mocha*, затем *LiveScript*, и в итоге — *JavaScript*.
-   **ECMAScript (ES)**: Это стандарт, по которому развивается язык. Современный JS часто называют **ES6+**, имея в виду огромный скачок в развитии, произошедший в 2015 году.

---

## 3. Почему стоит учить JavaScript?

1.  **Низкий порог входа**: Вам нужен только браузер и текстовый редактор, чтобы начать писать код.
2.  **Огромное сообщество**: На любой ваш вопрос уже есть ответ на Stack Overflow или GitHub.
3.  **Востребованность**: JS — самый популярный язык по версии GitHub Octoverse и Stack Overflow Developer Survey.
4.  **Универсальность**: Выучив один язык, вы сможете стать Full-stack разработчиком.

---

## 4. Особенности языка

-   **Динамическая типизация**: Тип переменной может меняться во время выполнения кода.
-   **Интерпретируемость**: Код не требует предварительной компиляции (в классическом понимании), он выполняется движком (например, V8 в Chrome) "на лету".
-   **Прототипное наследование**: Особенный способ передачи свойств между объектами.
