# Модуль 3: Арифметические и поразрядные операции

Этот модуль посвящен инструментам для работы с числами: от базовых математических действий до манипуляций на уровне отдельных битов.

---

## 1. Арифметические операции

Арифметические операторы принимают числовые значения (или значения, которые можно привести к числам) и возвращают один числовой результат.

### Базовые операторы
-   `+` (Сложение): `5 + 2 // 7` (Внимание: если один операнд — строка, произойдет конкатенация!)
-   `-` (Вычитание): `5 - 2 // 3`
-   `*` (Умножение): `5 * 2 // 10`
-   `/` (Деление): `5 / 2 // 2.5`
-   `%` (Остаток от деления): `5 % 2 // 1`
-   `**` (Возведение в степень): `5 ** 2 // 25` (Эквивалент `Math.pow(5, 2)`)

### Инкремент и Декремент
-   `++` (Увеличение на 1): `x++` или `++x`
-   `--` (Уменьшение на 1): `x--` или `--x`
-   *Разница*: `++x` (префиксный) возвращает значение **после** увеличения, `x++` (постфиксный) — **до**.

---

## 2. Операторы присваивания

Позволяют сократить запись арифметических операций:
-   `x += 5` (вместо `x = x + 5`)
-   `x *= 2` (вместо `x = x * 2`)
-   И аналогично для всех остальных: `-=`, `/=`, `%=`, `**=`.

---

## 3. Поразрядные (Битовые) операции

Битовые операторы рассматривают операнды как 32-битные последовательности нулей и единиц. Они работают непосредственно с двоичным представлением чисел.

| Оператор | Название | Описание | Пример (десятичный) |
| :--- | :--- | :--- | :--- |
| `&` | **И (AND)** | 1, если оба бита равны 1 | `5 & 1` (0101 & 0001) = `1` |
| `\|` | **ИЛИ (OR)** | 1, если хотя бы один бит равен 1 | `5 \| 2` (0101 \| 0010) = `7` (0111) |
| `^` | **Исключающее ИЛИ (XOR)** | 1, если только один из битов равен 1 | `5 ^ 1` (0101 ^ 0001) = `4` (0100) |
| `~` | **НЕ (NOT)** | Инвертирует все биты (превращает 0 в 1 и наоборот) | `~5` = `-6` |
| `<<` | **Левый сдвиг** | Сдвигает биты влево, добавляя нули справа | `5 << 1` = `10` (умножение на 2) |
| `>>` | **Правый сдвиг** | Сдвигает биты вправо, сохраняя знак | `5 >> 1` = `2` (деление на 2) |
| `>>>` | **Беззнаковый сдвиг** | Сдвигает вправо, заполняя нулями слева | `-5 >>> 1` = `2147483645` |

---

## 4. Приоритет операторов (Precedence)

Важно понимать, в каком порядке выполняются действия:
1. Группировка `( )`
2. Возведение в степень `**`
3. Умножение, деление и остаток `*`, `/`, `%`
4. Сложение и вычитание `+`, `-`
5. Битовые сдвиги `<<`, `>>`, `>>>`
6. Битовые И/ИЛИ/XOR `&`, `^`, `|`
7. Присваивание `=`

---

## 5. Практическое применение битовых операций

Хотя в повседневной веб-разработке они встречаются редко, они незаменимы для:
1.  **Работы с флагами доступа**: использование одного числа для хранения нескольких состояний (например, права `READ`, `WRITE`, `EXECUTE`).
2.  **Оптимизации**: быстрые математические расчеты в графике или криптографии.
3.  **Округления**: трюк `~~x` или `x | 0` позволяет быстро отбросить дробную часть числа.

```javascript
let x = 5.9;
console.log(~~x); // 5
```

---

## Резюме
- Арифметические операции интуитивно понятны, но помните о приоритетах.
- Инкремент бывает префиксным и постфиксным — это влияет на результат выражения.
- Битовые операции работают с 32-битными целыми числами.
- Поразрядные операторы могут быть полезны для компактного хранения настроек или быстрой математики.
