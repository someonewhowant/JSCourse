### Вопрос 4
В чем разница между `==` и `===`?

A) `==` сравнивает только объекты, `===` сравнивает всё  
B) `==` выполняет неявное приведение типов, `===` сравнивает без приведения типов  
C) Между ними нет разницы в современном JavaScript  
D) `===` работает быстрее, чем `==`  

---

### Вопрос 5
Какой тип данных используется для работы с целыми числами произвольной точности, выходящими за пределы безопасного диапазона Number?

A) `Double`  
B) `BigInt`  
C) `Integer`  
D) `Symbol`  

---

## Ответы

<details>
<summary>Нажмите, чтобы увидеть ответы</summary>

1. **C) "object"** (Это известная ошибка в JavaScript, которая сохраняется для обратной совместимости).
2. **C) []** (Пустые массивы и объекты в JavaScript считаются истинными значениями).
3. **A) 51** (Сначала `5 + "2"` дает строку `"52"`, затем `"52" - 1` приводит строку к числу и вычитает 1).
4. **B) == выполняет неявное приведение типов, === сравнивает без приведения.**
5. **B) BigInt** (Специальный тип для работы с очень большими числами).

</details>
