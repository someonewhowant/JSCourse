# Модуль 5: Преобразование данных

В этом модуле мы разберем, как превращать данные из одного типа в другой — как вручную, так и автоматически.

---

## 1. Явное преобразование (Explicit Conversion)

Это когда мы намеренно меняем тип данных с помощью встроенных функций.

### В строку: `String()`
```javascript
String(123);     // "123"
String(true);    // "true"
String(null);    // "null"
```

### В число: `Number()`
```javascript
Number("456");   // 456
Number("456.7"); // 456.7
Number("abc");   // NaN (Not a Number)
Number(true);    // 1
Number(false);   // 0
Number(null);    // 0
Number(undefined); // NaN
```

### В логическое значение: `Boolean()`
Помним список "falsy" значений из Модуля 2 (`0`, `""`, `null`, `undefined`, `NaN`). Все остальное — `true`.

---

## 2. Парсинг чисел: `parseInt` и `parseFloat`

В отличие от `Number()`, эти функции "читают" строку слева направо, пока не встретят нечисловой символ.

```javascript
parseInt("100px");   // 100
parseFloat("12.5em"); // 12.5
parseInt("abc10");    // NaN (так как начинается не с числа)

// parseInt принимает второй аргумент — систему счисления (основание)
parseInt("11", 2);    // 3 (двоичная система)
parseInt("FF", 16);   // 255 (шестнадцатеричная система)
```

---

## 3. Неявное приведение (Type Coercion)

JavaScript часто пытается "помочь", автоматически меняя типы при операциях.

- **Сложение со строкой**: Если хоть один операнд — строка, всё становится строкой.
  `5 + "10" // "510"`
- **Математические операции (кроме +)**: Всегда пытаются привести операнды к числам.
  `"10" - 5 // 5`
  `"6" * "2" // 12`
- **Логический контекст**: В `if`, `while`, `&&`, `||` значения приводятся к `Boolean`.

---

## 4. Преобразование объектов в JSON

JSON (JavaScript Object Notation) — основной формат обмена данными в вебе.

1.  **`JSON.stringify(obj)`**: Превращает объект/массив в строку JSON (сериализация).
2.  **`JSON.parse(str)`**: Превращает строку JSON обратно в объект/массив (десериализация).

```javascript
const user = { id: 1, name: "Alice" };

const jsonString = JSON.stringify(user); 
console.log(jsonString); // '{"id":1,"name":"Alice"}'

const parsedUser = JSON.parse(jsonString);
console.log(parsedUser.name); // "Alice"
```

---

## 5. Быстрые "хаки" преобразования

В реальном коде часто встречаются короткие записи:
- **В число**: Унарный плюс `+`
  `+"42" // 42`
- **В логический тип**: Двойное отрицание `!!`
  `!!"hello" // true`
- **В строку**: Сложение с пустой строкой
  `5 + "" // "5"`

---

## Резюме
- Используйте `String()`, `Number()` и `Boolean()` для явности.
- `parseInt` и `parseFloat` полезны для извлечения чисел из строк с единицами измерения.
- Всегда будьте осторожны с неявным приведением при использовании оператора `+`.
- JSON — стандарт для передачи данных, используйте `stringify` и `parse`.
