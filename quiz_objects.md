# Квиз: Объекты в JavaScript

Проверьте свои знания об объектах и их методах. Ответы в конце.

---

### Вопрос 1
Как правильно удалить свойство `age` из объекта `user`?

A) `user.age = null`  
B) `delete user.age`  
C) `user.remove("age")`  
D) `user.age = undefined`  

---

### Вопрос 2
Что выведет этот код?

```javascript
const user = { name: "Anna" };
const admin = user;
admin.name = "Ivan";
console.log(user.name);
```

A) `"Anna"`  
B) `"Ivan"`  
C) `undefined`  
D) Ошибка  

---

### Вопрос 3
Какой метод возвращает массив всех **значений** свойств объекта?

A) `Object.keys()`  
B) `Object.entries()`  
C) `Object.values()`  
D) `Object.get()`  

---

### Вопрос 4
На что ссылается ключевое слово `this` внутри метода объекта при обычном вызове?

A) На глобальный объект `window`  
B) На саму функцию  
C) На объект, вызвавший метод  
D) На `undefined`  

---

### Вопрос 5
Для чего используется оператор `?.` (опциональная цепочка)?

A) Для сравнения двух объектов  
B) Для быстрого создания копии объекта  
C) Для безопасного доступа к свойствам объекта, которые могут быть `null` или `undefined`  
D) Это синтаксическая ошибка  

---

## Ответы

<details>
<summary>Нажмите, чтобы увидеть ответы</summary>

1. **B) delete user.age** (Специальный оператор для удаления свойств).
2. **B) "Ivan"** (Объекты копируются по ссылке, поэтому изменения через `admin` видны в `user`).
3. **C) Object.values()**.
4. **C) На объект, вызвавший метод**.
5. **C) Для безопасного доступа к свойствам...** (Предотвращает ошибку "Cannot read property of undefined").
